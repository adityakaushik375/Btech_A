CSS Grid is a 2-dimensional layout system (rows + columns) used to build responsive webpages easily. Flexbox → 1D (row or column) Grid → 2D (rows and columns) Grid Container To start Grid: display : grid; Important Properties of Grid Container 1. grid-template-columns Defines the number & width of columns. grid-template-columns: 200px 200px 200px; grid-template-columns: repeat(3, 200px); grid-template-columns: 1fr 2fr 1fr; 2. grid-template-rows Defines row heights. grid-template-rows: 100px 100px; 3. gap / column-gap / row-gap Space between grid items. gap: 20px; row-gap: 10px; column-gap: 30px; Grid Item Properties These properties are used on children. 1. grid-column Span columns. grid-column: 1 / 3; /* from col 1 to 3 */ grid-column: span 2; /* take 2 columns */ 2. grid-row Span rows. grid-row: span 3; 3. justify-self (horizontal) 4. align-self (vertical)
